# ChatGPT Clone

A high-performance AI chat interface built with React and Node.js, featuring a modern, sleek UI inspired by OpenAI's ChatGPT.

## Features

- **Real-time Chat**: Seamless interaction with a backend AI service.
- **Dynamic Input**: Auto-resizing textarea for better user experience.
- **Modern UI**: Clean, minimal design with professional typography.
- **Conversation History**: Local state management for current chat sessions.
- **Responsive Design**: Optimized for different screen sizes.

## Tech Stack

- **Frontend**: React, Vite, CSS3 (Vanilla)
- **Backend**: Node.js, Express
- **API**: OpenAI API (configured in server)

## Setup Instructions

### 1. Backend Setup

1. Navigate to the `Server` directory.
2. Install dependencies: `npm install`.
3. Add your API keys in the server configuration.
4. Start the server: `node index.js`.

### 2. Frontend Setup

1. Navigate to the root of `ChatGPT`.
2. Install dependencies: `npm install`.
3. Start the dev server: `npm run dev`.
